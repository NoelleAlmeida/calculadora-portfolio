Sobre o projeto

Esta calculadora foi desenvolvida como um projeto de portfólio para praticar HTML, CSS e JavaScript, unindo lógica de programação com cuidado visual e experiência do usuário.

O projeto começou como uma calculadora básica e foi evoluindo para uma versão mais completa, com melhorias visuais, novas funcionalidades e uma calculadora científica integrada.

Funcionalidades do projeto

  • Operações matemáticas básicas
  • Modo científico com funções extras
  • Porcentagem
  • Botão de apagar um caractere por vez
  • Suporte ao teclado
  • Tema claro e escuro com salvamento no navegador
  • Página Sobre integrada ao projeto
  • Tratamento de erro para divisão por zero
  • Layout responsivo

Modo científico

A calculadora também possui um modo científico, com funções como potência, raiz quadrada, seno, cosseno, tangente, logaritmo, logaritmo natural, valor absoluto, uso de parênteses e número π.

Tecnologias usadas

  • HTML
  • CSS
  • JavaScript
  • Git e GitHub
  • Vercel

O que aprendi

  • Estruturar páginas web do zero
  • Separar HTML, CSS e JavaScript de forma organizada
  • Manipular o DOM e eventos de clique e teclado
  • Criar temas visuais com persistência usando localStorage
  • Evoluir um projeto simples para uma versão mais completa
  • Versionar, publicar e atualizar um projeto online
