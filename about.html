Calculadora Sobre

Sobre o projeto

Esta calculadora foi desenvolvida como um projeto de portfólio para praticar HTML, CSS e JavaScript.

Durante o desenvolvimento, trabalhei habilidades como estruturação de páginas, estilização responsiva, lógica de programação, manipulação do DOM e uso de Git/GitHub.

Também adicionei melhorias para deixar a experiência mais completa, como suporte ao teclado, botão de apagar, alternância entre tema claro e escuro e um layout mais moderno.

Tecnologias usadas

  • HTML
  • CSS
  • JavaScript
  • Git e GitHub

O que aprendi

  • Criar uma interface web do zero
  • Separar estrutura, estilo e lógica em arquivos diferentes
  • Trabalhar com eventos de clique e teclado
  • Versionar e publicar um projeto
